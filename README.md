# EPS Worldwide - Global Logistics & Courier Services

A professional website for EPS Worldwide, offering domestic and international courier, cargo, and third-party logistics services.

## Features

- **AI Chat Support** - 24/7 chatbot for customer inquiries
- **User Dashboard** - Track shipments and manage preferences
- **Referral Program** - Earn rewards by referring friends
- **Service Comparison** - Compare with competitors
- **Blog & News** - Industry insights and updates
- **Testimonials** - Real customer reviews
- **Mobile Responsive** - Works on all devices
- **Volumetric Weight Calculator** - Calculate shipping costs
- **Secure Forms** - Newsletter signup, contact forms, booking

## Services

- Domestic Courier
- International Export & Import
- Warehousing & Storage
- Third-Party Logistics (3PL)

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Or deploy to Vercel for live hosting

## Deployment

### Deploy to Vercel (Recommended)
1. Go to https://vercel.com/new
2. Import this GitHub repository
3. Click Deploy

Your site will be live at `https://eps-worldwide.vercel.app`
